# AntiDroneSimulator

## 1. 玩家设计

### 1.1 基础操作

* 移动
  *  [W]	[A]	[S]	[D]
* 转动视角
  * [MouseX]	[MouseY]
* 开火
  * [Mouse 0]
* 装弹
  * [Mouse 2]
* 跳跃
  * [Space]
* 翻滚
  * [Mouse 1]
* 蹲下、起立
  * [LeftControl]



## 2. 无人机设计

### 2.1 基础操作

* 飞行
  * [W]	[A]	[S]	[D]	[Q]	[E] 
* 发射导弹
  * [F]

### 2.2 无人机控制算法

* 静止
* 随机移动
* Flocking 算法



## 3. 技能设计

* 切换控制
  * [Tab]
* 时间暂停
  * [P]
* 观察者模式
  * [O]
